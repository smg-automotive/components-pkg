# Discrete Slider

DiscreteSlider is a custom component built utilizing the Chakra UI Slider component.
This slider allows for discreet selections represented by marks, that reflect stepped values.
This can be useful in scenarios where you want to offer a small selection of options on a slider for a user to pick from.
